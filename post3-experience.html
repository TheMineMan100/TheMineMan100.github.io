I have some previous programming experience, since I experimented with making a simple game using C++. While I did eventually halt the project, it did give helpful insight in the basic structure of code, along with some basic bugfixing skills.

In order to attempt the creation of that game I worked through Michael Dawson's “Beginning C++ Through Game Programming”, which is a very informative book on the subject. I also took the online workshop “C++ Fast Track for Games Programming”, and of course did a good deal of research outside of this.
